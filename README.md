# Smart File Compression Tool

## Description
A C++ application that compresses and decompresses files using Huffman Coding.

## Features
- Compress text files
- Decompress text files
- Show compression statistics
- Lossless compression

## Technologies
- C++17
- Git
- CMake
- Visual Studio Code
